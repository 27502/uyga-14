JavaScript Concepts

What is a Function?

A function is a block of organized, reusable code that performs a single, specific task. In JavaScript, functions are fundamental tools for structuring, managing, and reusing code. Functions help break a program into manageable parts, making it easier to debug, maintain, and extend.

Key Features of Functions


Modularity: Functions allow us to compartmentalize code, making programs easier to understand and work on.

Reusability: Once defined, a function can be called multiple times, reducing repetition and code size.

Scoping: Functions create their own scope, which helps in controlling variable access and avoiding name conflicts.

Functions

Functions allow us to create reusable code blocks that can be called whenever needed. They are a core part of programming.

function greet() {

   console.log("Hello, World!");
}

Declaring a Function

You can declare a function using the function keyword, followed by the function name, parameters, and a code block:

function greet(name) {

console.log("Hello, " + name + "!");
}

To call the function and pass in an argument:

greet("Alice");

Function Parameters and Arguments

Functions can accept parameters (placeholders for values) when they are declared and receive arguments (actual values) when they are called.

function add(a, b)
{
return a + b;
}
let sum = add(5, 3);

Date

The Date object is used to work with dates and times. It can provide current date/time and be formatted as needed.

let today = new Date();

console.log(today);

Specific Date and Time:

let specificDate = new Date(2023, 10, 11, 14, 0, 0);

Using a Timestamp:

let dateFromTimestamp = new Date(1681106400000)

Getting Date and Time Components:

The Date object provides methods to retrieve date and time components individually:

let year = currentDate.getFullYear();

let month = currentDate.getMonth();

let day = currentDate.getDate();
